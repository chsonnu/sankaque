Sankaque

Sankaque is a multi-threaded, priority queue server.

How does Sankaque differ from other message queues such as Beanstalk, Sidekiq, RabbitMQ, etc.?

The main difference is that every message requires a priority, and that priority determines where in the queue the message ends up.

How fast is it?

Sankaque is written in C and the priority queue is implemented with a binary heap, which means the insert/delete operations' worst case time complexities are O(log n).

Use Cases

  • Prioritizing resource intensive background tasks.
  • Simplifying queue architectures that emulate a priority queue (usually by polling multiple queues).

Install It

$ wget https://github.com/chsonnu/sankaque/releases/tag/sankaque-0.1.0.tar.gz
$ tar xzf sankaque-0.1.0.tar.gz
$ cd sankaque-0.1.0
$ make

Run It

$ ./sankaque -l 0.0.0.0 -p 12000
